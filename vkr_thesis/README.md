# Текст ВКР

**[VKR_Biksalin_final.pdf](VKR_Biksalin_final.pdf)** — итоговый текст ВКР Биксалина Артёма Ильнуровича, 94 страницы, прошедший проверку в LMS НИУ ВШЭ.

## Содержание

- **Введение** — постановка задачи: автоматизация анализа обратной связи в логистической e-commerce компании на основе LLM
- **Глава 1** — выявление бизнес-проблемы, PEST/SWOT, обзор подходов от TF-IDF до LLM, информационная безопасность, TOWS-матрица
- **Глава 2** — проектирование ИАС: требования, архитектура (ArchiMate, C4, UML), модель TO-BE, экономическое обоснование, риски, KPI, технологический стек
- **Глава 3** — реализация конвейера, методика сравнительного эксперимента, результаты (TF-IDF + XGBoost, YandexGPT Pro/Embeddings/Classifier, Qwen 2.5 7B, DeepSeek LLM 7B, DeepSeek R1 7B), промпт-инжиниринг
- **Заключение** — итоги по 6 задачам, проверка гипотезы, направления развития
- **Приложения А–Г** — пример отчёта LLM, скриншоты дашборда, системные промпты, скриншоты Telegram-бота
