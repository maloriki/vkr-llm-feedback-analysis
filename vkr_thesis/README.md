# Тексты ВКР

Эта папка содержит итоговые документы выпускной квалификационной работы.

## Содержимое

- **`VKR_Biksalin.docx`** — основной документ ВКР: 70+ страниц, 10 диаграмм,
  7 таблиц, 3 приложения, 46 источников.

- **`VKR_Status_Report.pdf`** — отчёт по статусу проекта:
  что сделано, сильные и слабые стороны, требования методичек ВШЭ,
  план до защиты.

## Структура VKR

- **Введение** — актуальность, цель, задачи, гипотеза, ограничения,
  методы, теоретическая база.
- **Глава 1** — анализ AS-IS, диаграммы ArchiMate / BPMN / Исикавы,
  обзор подходов (TF-IDF → BERT → LLM).
- **Глава 2** — требования, архитектура (C4, ArchiMate App), TO-BE,
  экономическое обоснование, риски, KPI.
- **Глава 3** — реализация конвейера, эксперимент с 6 моделями,
  результаты, промпт-инжиниринг.
- **Заключение** — выводы по структуре «задача → результат → значение».
- **Приложения** — пример отчёта LLM, скриншоты дашборда, полные тексты
  системных промптов.

## Связь с кодом

Реализация системы (LLM-конвейер, Streamlit-дашборд, Telegram-бот),
а также скрипты экспериментов и предсказания моделей находятся
в корне репозитория.
